# Recipe Finder

A Recipe Finder web app built with React + Vite + Tailwind that uses TheMealDB to search recipes by name/ingredient, view details, and save favorites.

## Features
- Search by name or ingredient (debounced)
- Category filters
- Recipe details page (ingredients + instructions)
- Favorites persisted to localStorage
- Responsive design with Tailwind CSS
- Routing with React Router

## Run locally
```bash
# install
npm install

# dev server
npm run dev
